# VAF Python Zero to Hero

Python for AI-native architects, vibe coders, and Velocity Architecture Academy practitioners.

Python is the automation and intelligence layer behind architecture tooling, decision scoring, artefact generation, prompt guardrails, and VAF decision flows.

## Run the course

```bash
python run_all.py
```

## Learning path

| Lesson | Title | Outcome |
|---|---|---|
| 00 | Programming Refresher | Revisit values, variables, output, and simple structure |
| 01 | Hello Python | Run the first Python file |
| 02 | Variables and Values | Work with text, numbers, and booleans |
| 03 | Functions | Create reusable decision instructions |
| 04 | Lists and Dictionaries | Model collections and structured records |
| 05 | JSON Models | Read VAF-style data from JSON |
| 06 | Files and Folders | Write generated outputs |
| 07 | CSV Data | Read tabular assessment data |
| 08 | API Basics | Understand API-shaped requests and responses |
| 09 | Decision Scoring Engine | Score value, risk, readiness, and confidence |
| 10 | Artefact Generator | Generate a markdown decision summary |
| 11 | Prompt Guardrails | Assess prompt readiness and review level |
| 12 | Capstone VAF Automation | Run a full VAF-style automation flow |

## Architecture lens

Dictionary = structured artefact. Function = decision rule. List = portfolio. JSON = exchange model. File output = generated artefact. Script = automation workflow.
